# EndlessPeak Client

A lightweight JavaScript + Pixi.js client for 2D games.

## Features

Rendering

* 2D rendering using Pixi.js
* Sprite management
* Basic UI elements

Networking

* WebSocket client for realâ€‘time communication

Build System

* npm-based workflow
* Development server
* Production bundling

## Requirements

* Node.js (LTS recommended)
* npm

## Getting Started

```bash
git clone git@github.com:kenka1/EndlessPeakClient.git
cd EndlessPeakClient
npm install
```

## Running the Client

Development mode:

```bash
npm run dev
```

Open in browser:

```
http://localhost:5173
```

## Build

```bash
npm run build
```

Output will appear in the `dist/` directory.

## Server

Simple visualization works with:

```
git@github.com:kenka1/EndlessPeakServer.git
```
